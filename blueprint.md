# Blueprint: 주식 시장 분석 서비스

## 개요

이 애플리케이션은 사용자가 주식 종목을 검색하고 해당 종목의 상세 정보와 시세 차트를 확인할 수 있는 웹 기반 주식 시장 분석 도구입니다.

## 스타일, 디자인, 기능

*   **UI/UX:**
    *   현대적이고 직관적인 다크 테마를 기반으로 합니다.
    *   사용자가 쉽게 종목을 검색하고 차트를 확인할 수 있는 단순한 레이아웃을 제공합니다.
    *   모바일 및 데스크톱 환경에 모두 대응하는 반응형 디자인을 적용합니다.
*   **핵심 기능:**
    *   **종목 검색:** 사용자가 주식 종목 코드나 이름을 입력하여 검색할 수 있는 입력 필드를 제공합니다.
    *   **상세 정보 표시:** 검색된 종목의 현재가, 등락률, 거래량 등 주요 정보를 표시합니다.
    *   **시세 차트:** `lightweight-charts` 라이브러리를 사용하여 검색된 종목의 시세(일봉)를 캔들스틱 차트로 시각화합니다.
*   **기술 스택:**
    *   **Frontend:** 순수 HTML, CSS, JavaScript (프레임워크 없음)
    *   **Charting:** TradingView Lightweight Charts
    *   **데이터:** 초기 버전에서는 실제 API 대신 모의(mock) 데이터를 사용하여 기능을 구현합니다. 향후 실제 주식 데이터 API로 쉽게 교체할 수 있도록 구조를 설계합니다.
*   **컴포넌트:**
    *   **헤더:** 애플리케이션 제목
    *   **검색 바:** 종목 코드 입력 필드 및 검색 버튼
    *   **정보 표시 영역:** 종목의 상세 정보 카드
    *   **차트 영역:** 시세 챠트

## 현재 계획

*   **목표:** 사용자가 주식 종목을 검색하고 차트를 볼 수 있는 동적 웹 애플리케이션 구축.
*   **진행 단계:**
    1.  `index.html`에 종목 검색을 위한 입력 필드와 버튼, 그리고 주식 정보를 표시할 영역을 추가합니다.
    2.  `style.css`를 수정하여 새로 추가된 UI 요소들에 대한 스타일을 적용하고, 전체적인 레이아웃을 개선합니다.
    3.  `main.js`를 대대적으로 리팩토링합니다.
        *   UI 요소(입력 필드, 버튼 등)에 대한 이벤트 리스너를 추가합니다.
        *   실제 API 호출을 시뮬레이션하는 비동기 함수를 만듭니다. 이 함수는 검색된 종목에 대한 모의 데이터를 반환합니다.
        *   '검색' 버튼 클릭 시, 모의 데이터 함수를 호출하고 받아온 데이터로 차트와 정보 영역을 동적으로 업데이트하는 로직을 구현합니다.
        *   코드를 모듈화하여 관리하기 용이하게 만듭니다.
